◆ buildDetailedMesh()

void pepr3d::Geometry::buildDetailedMesh ( )
private

Build a CGAL mesh over detailed triangles.

Yes, we are about to hash floating point values. These values come from CGAL exact kernel, so they should be bit-equal and safe to hash. There is no betters way to get indices before this, as different color parts are stored in different polygons.
